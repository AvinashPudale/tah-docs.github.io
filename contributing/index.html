Contributing

The source code for all the Tah smartphone apps is open source and is hosted on GitHub. If you are looking to contribute to our efforts of making Tah an awesome platform for building amazing things using Bluetooth, join us !

Apps

Each app has been developed for iOS and Android, though some are still in the works. We have a git repo on GitHub for each app, with the iOS and Android source codes as git submodules.

Submodules allow you to keep a Git repository as a subdirectory of another repository

So for example, you'll find the source code for the Tah-iOS and the Tah-Android apps inside a repository called Tah.

Sketches

You will notice that the above repository also has a folder called ArdSCL. This consists of the sketch that will be uploaded on the Tah when using the Tah app (iOS or Android).

Begin Contributing

To contribute begin by forking the repo that you want to contribute to.

  • Click on the repository that you want to contribute to :
  • Fork the repository, I'll be contributing to the Tah-iOS app in this example :

  • A copy of this repository will be created in your account. You can now make changes to it.

  • Clone this repo and you can make changes to the source code.

  • Once you have made the changes, you can send a pull request. Click the marked button on your repo :
