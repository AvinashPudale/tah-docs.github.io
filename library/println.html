size_t (long) println(val, format)

Description

Prints data to the serial port as human-readable ASCII text followed by a carriage return character (ASCII 13, or '\r') and a newline character (ASCII 10, or '\n'). This command takes the same forms as Serial.print() in Tah and myTAH.print().

Syntax

myTAH.println(val)
myTAH.println(val, format)

Parameters

val: the value to print - any data type
format: specifies the number base (for integral data types) or number of decimal places (for floating point types)

Returns

size_t (long): println() returns the number of bytes written, though reading that number is optional

Example

/*
Analog input
reads an analog input on analog in 0, prints the value out.
*/

TAH myTAH;
int analogValue = 0;    // variable to hold the analog value

void setup()
{
// open the serial port at 9600 bps:
Serial.begin(9600);
myTAH.Serial(9600);         //Open the Tah Serial port at 9600 bps
myTAH.enterCommandMode();   // Enters TAH command mode
myTAH.setName("TAH");
myTAH.setWorkRole(SLAVE);
myTAH.setAuth(OPEN);
} 

void loop()
{
// read the analog input on pin 0:
analogValue = analogRead(0);

// print it out in many formats to Tah BLE port:

myTAH.println(analogValue);       // print as an ASCII-encoded decimal
myTAH.println(analogValue, DEC);  // print as an ASCII-encoded decimal
myTAH.println(analogValue, HEX);  // print as an ASCII-encoded hexadecimal
myTAH.println(analogValue, OCT);  // print as an ASCII-encoded octal
myTAH.println(analogValue, BIN);  // print as an ASCII-encoded binary

// delay 10 milliseconds before the next reading:
delay(10);
}
