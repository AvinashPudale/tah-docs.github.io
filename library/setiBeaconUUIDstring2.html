void setiBeaconUUIDstring2(char UUID2[])

Description

Every iBeacon device has 20 bytes of UUID that is divided into 4 parts. This utility will set the second part of the UUID.

Syntax

setiBeaconUUIDstring2(UUID2);

Parameters

UUID2 - pass parameter of data type - string

Returns

None
