void setBatteryMonitorSwitch(bool status)

Description

This function used to set the battery monitor switch as on or off.It gives the status of battery when it is on.


Syntax
setBatteryMonitorSwitch(status);

Parameters
status: takes the parameters as bool
  1 = ON
  0 = OFF

Returns
Nothing
