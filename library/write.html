virtual size_t write(uint8_t byte)

Description

Writes binary data to the Tah BLE serial port. This data is sent as a byte or series of bytes; to send the characters representing the digits of a number use the print() function instead.


Syntax
myTAH.write(val);
myTAH.write(str); 
myTAH.write(buf, len);

Parameters
val: a value to send as a single byte
str: a string to send as a series of bytes
buf: an array to send as a series of bytes
len: the length of the buffer

Returns
byte - write() will return the number of bytes written, though reading that number is optional.

Example
TAH myTAH;
  void setup()
  {
    Serial.begin(9600);
    myTAH.begin(9600);            //Tah BLE serial port 
    myTAH.enterCommandMode();     // Enters TAH command mode
    myTAH.setName("TAH");
    myTAH.setWorkRole(SLAVE);
    myTAH.setAuth(OPEN);
  }   
  
  void loop()
  {  
    myTAH.write(45);                        // send a byte with the value 45
    int bytesSent = myTAH.write(“hello”);   //send the string “hello” and return the length of the string.
  }
