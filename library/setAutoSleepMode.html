setAutoSleepMode()

Description

When there is no data transmition between BLE modules the BLE will go into auto sleep mode to save on power consumption. This function puts the BLE in an auto sleep mode when there is no data transfer. In deep sleep mode the Tah BLE consumes upto 6mA of current.

Syntax

setAutoSleepMode();

Parameters

mode: pass the int argument to set the auto sleep mode.
0:Auto sleep
1:Don't auto sleep
Default: 1

Returns

None
