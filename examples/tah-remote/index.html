Tah Remote

Tah is a Bluetooth Low Energy open source development platform compatible with the Arduino.

The Tah has an IR shield which can be used to control infrared devices in a room. In this example we are going to see how to use the Tah write app, which is basically a serial monitor, over Bluetooth from a smartphone. We are going to send commands to control a Toshiba TV

Requirements

  • Tah
  • IR shield
  • Tah write app

Step 1

Connect the Tah board to the USB port and open the Arduino IDE. Download the Tah library and rename it to Tah after extracting.

Step 2

Upload the Tah remote Arduino sketch from here.

Step 3

Use the Tah write app to communicate with the Tah board. You can find the source for it here. Valid inputs (in this case we are going to control the power and channel numbers for a Toshiba TV)

  • p = power on/off
  • [1-9] = channel number
