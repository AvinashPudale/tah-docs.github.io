Sensors & Relays

Tah is a Bluetooth Low Energy open source development platform compatible with the Arduino.

It can be used as a smart sensor node by interfacing the sensors with the Sensor + Relay Shield. By interfacing 6 different sensors on the shield, we can develop a real time data acquisition application with a smart phone. We could interface different sensors like sonar, temperature, touch, LDR, rain, wind, PIR motion, soil moisture etc.

Requirements
  • Tah
  • Tah Relay + Sensor Shield
  • Wire Stripper
  • Connecting Wires
  • 12V DC Adapter
  • Arduino IDE installed
  • iPhone 4S or newer with the Tah Sensor app installed

Step 1

To get started with this demo download the Tah Arduino Library and install it into your existing Arduino IDE.

Download the Tah Sensor sketch and upload it to Tah.

Step 2

Short the echo and the trigger pin of the sonar sensor and connect it to any digital I/O of the Tah on the relay + sensor shield. Here we are using digital pin number 5. Similarly, connect the VCC and the GND wire on the relay + sensor shield. Plug the Tah on the shield and then power up the Tah sensor + relay shield.


Step 3

  • Open the Tah Sensor app on your iPhone
  • Press 'Scan' inside the Tah Sensor app to scan for Bluetooth Devices
  • Connect to the device named 'Tah Sensor'
  • Select the 'Sonar Sensor' tab
  • Go to 'Settings' and select the pin used for the sensor.

Step 4

See the change in value of distance in the Tah Sensor app
